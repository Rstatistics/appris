The "Query" section shows a panel with information about the query gene.

The showed information depends on whether the type of query:

  • When the query is a gene identifier/name (from APPRIS Database or from APPRIS WebServer), the panel contains the name and the identifier of the gene, the species name, and the genome location of the gene and the assembly version. Optionally, the classification (biotype) of the gene could be appear if the gene ID comes from Ensembl.

  • When the query input is protein sequences (only from APPRIS WebServer), the panel contains one identifier of the sequence and the species name.
