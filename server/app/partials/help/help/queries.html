The APPRIS WebServer accepts two kind of queries:

  • Alternative protein sequences from the same gene,

  • Or the gene name (or EnsemblID) from a specific Assembly version

    • In both cases, the server requires the name of the species from the list of vertebrate species. If the alternative sequences are from a species that is not in this list the WebServer will still work, simply select the most similar species.

An Status Log panel appears after a submitted a job, indicating whether the job is pending, running, finished or gave an error.
