Data
====
In this directory we save the *__raw outputs__* of APPRIS per gene.


Directory structure
===================
This directory contains the following subdirectories:

```
{APPRIS_version}
        |
        |___ {species_name}
                    |
                    |__ {datasource_version}
                                |
                                |__ {chromosome/gene_id subdirectories}
                                                |
                                                |__ {APPRIS_output_files}
```
